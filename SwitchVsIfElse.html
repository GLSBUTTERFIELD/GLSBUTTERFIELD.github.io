What is the difference between switch and if/else? When would you use one versus the other?

Both switch and if/else are conditional statements that execute code depending on whether the condition(s) is/are met.

The main difference between the two is that switch statements evaluate one variable (such as 'char,' 'string', etc.) while if/else statements can evaluate multiple variables, meaning you can require multiple conditions to be met with if/else statements.

I would use switch when I have one variable that can be evaluated against a specific set of answers, and I would use if/else when there are multiple variables or conditions to be met.
